# Release notes prompt

Use this prompt when preparing a release.

```text
Read CHANGELOG.md, recent commits, AGENTS.md, and docs/specs/.

Write release notes for VerdictCI <version>.

Include:
- new capabilities;
- breaking changes;
- migration notes;
- CLI changes;
- config schema changes;
- result JSON changes;
- known limitations.

Keep it concise and developer-focused.
Do not claim SaaS, GitHub App, or hosted features unless they exist.
```
